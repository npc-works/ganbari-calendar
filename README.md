# がんばりカレンダー

子供の毎日の学習・練習を記録し、がんばった分だけゲーム時間がたまるカレンダーアプリ。

🌐 **本家URL（常に最新版・PWA対応）**: <https://n-pc.jp/ganbari-calendar/>

📱 スマホで開いて「ホーム画面に追加」するとアプリのように使えます。

---

## 主な機能

- 必須タスク（ピアノ・しゅくだい等）をすべてクリアしないとゲーム時間入力が解放されない
- プリント枚数 → ゲーム時間自動換算（例: 国語1まい=10分）
- ゲーム時間消費・お小遣い交換
- 編集ロック（パスワードなしでは値を減らせない／追加はOK）
- 複数の子供をタブで切替（データは独立）
- 自動バックアップ（指定フォルダへ1日1回JSON書出し・履歴削除でも復元可）
- 🆕 **通知機能**: 記録保存時に親のメール or スマホPush通知（任意設定）
- 🆕 **PWA対応**: ホーム画面追加・オフライン動作

---

## 使い方

### 通常利用（推奨）
ブラウザで <https://n-pc.jp/ganbari-calendar/> を開いて使うだけ。常に最新版が動きます。

### ホーム画面に追加（PWA）
- **iPhone Safari**: 共有ボタン → 「ホーム画面に追加」
- **Android Chrome**: メニュー → 「ホーム画面に追加」

### 設定変更
右上の歯車（⚙）→ パスワード入力で設定画面を開けます。

| 項目 | できること |
|------|-----------|
| こども管理 | 子供の追加・名前変更・削除 |
| 必須チェック項目 | 名前・追加・削除（ゲーム解放の条件） |
| プリント項目 | 名前・1枚あたりの分数・単位（まい/個/セット等）・追加・削除 |
| こうかんレート | 交換に必要な分数と金額 |
| パスワード | 4桁の数字で変更 |
| 通知せってい | メール／ntfy.sh Pushの送信先を設定 |
| じどうバックアップ | 指定フォルダへ自動バックアップ |

## 通知機能の使い方

子供が記録を保存・更新するたびに、親のメール／スマホへ通知が届きます。

### メール通知（簡単・アプリ不要）
1. 設定画面の「通知せってい」を開く
2. 「通知をつかう」をON
3. メールアドレス欄に通知先メアドを入力
4. 「ほぞん」

### スマホPush通知（即時・無料アプリ要）
1. スマホに **ntfy.sh** アプリをインストール（[ntfy.sh](https://ntfy.sh/) から無料DL）
2. アプリで好きな合言葉（例: `ganbari-tarou-7k3p9q`）を「Subscribe」
3. 設定画面の「通知せってい」のntfyトピック名欄に同じ合言葉を入力
4. 「ほぞん」

両方入力すれば両方に届きます。

### 通知のON/OFFは「通知するイベント」で個別に切替可能
- 毎日のきろくをほぞんしたとき
- ゲーム時間をついかしたとき
- お小遣いに交換したとき

## ローカル単独運用（オフライン専用）

技術者向け。リリースzipをDLして展開、`index.html` をブラウザで開けば動きます。

⚠️ ローカル単独だとメール通知は動きません（自前のAPIエンドポイントが必要）。スマホPush（ntfy.sh）はローカルでも動きます。

## 自前ホスティング（技術者向け）

```
/your-server/ganbari-calendar/
├── index.html
├── config.local.js          ← config.local.js.example をコピーして実値入れる
├── manifest.json
├── sw.js
├── favicon.ico
├── icons/
├── api/
│   ├── notify.php
│   ├── config.php           ← config.php.example をコピーして実SMTP情報入れる
│   ├── .htaccess
│   └── data/.htaccess
└── vendor/                  ← composer install で再現可能（同梱もOK）
```

### セットアップ手順
1. 静的ファイル（`index.html` 等）はそのまま配置
2. `api/config.php.example` を `api/config.php` にコピーし、SMTP情報・共有トークンを入力
3. `config.local.js.example` を `config.local.js` にコピーし、APIエンドポイントURLとトークンを入力（**`api/config.php` の shared_token と一致必須**）
4. PHP 8.0+ が動くサーバーへアップロード（PHPMailer同梱）

### スパム対策（標準装備）
- Origin/Refererチェック
- Honeypot
- 共有秘密トークン
- レート制限（IP単位 1分1通／宛先メール単位 1時間20通）
- 停止スイッチ（`api/kill-switch` ファイルを置けば全リクエスト503）

## 技術仕様

- **フロントエンド**: HTML/CSS/JavaScript（外部ライブラリ依存なし）
- **データ保存**: localStorage（外部送信なし・各ブラウザ内）
- **通知メール**: PHP 8.0+ ＋ PHPMailer（SMTPS）
- **通知Push**: ntfy.sh（無料公開サーバー）
- **PWA**: manifest.json + Service Worker（オフラインキャッシュ）
- **動作環境**: モダンブラウザ（Safari・Chrome・Firefox・Edge）
